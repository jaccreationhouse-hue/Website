# Homepage Marquees Design

## Goal

Restore the What You Can Expect marquee and convert Industries We Serve into
an icon-pill marquee while leaving Our Highlights and all other sections
unchanged.

## Approved Design

- What You Can Expect uses text pills and scrolls left.
- Industries We Serve uses icon-and-label pills and scrolls right.
- Both marquees pause on hover and keyboard focus.
- Reduced-motion users see static, wrapping pill rows.
- Existing colors, typography, and homepage order remain unchanged.
