# Marquee Style Consistency Design

## Goal

Make What You Can Expect visually consistent with the Industries We Serve
marquee while preserving opposite scroll directions.

## Approved Design

- Both marquees use the same icon-pill structure and shared classes.
- Both use identical height, padding, gap, radius, typography, borders, hover,
  focus, fade edges, speed, and reduced-motion behavior.
- Expectations scroll left; Industries scroll right.
- Only labels and icons differ.
