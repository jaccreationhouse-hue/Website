# What We Do Section Design

## Goal

Redesign only the homepage What We Do section for stronger UI/UX consistency
with the updated Industries and Highlights sections.

## Approved Design

- Use the shared centered chip, heading, and supporting-copy pattern.
- Preserve all existing services, icons, and routes.
- Make each complete service card a link.
- Add outcome-focused descriptions and a clear arrow affordance.
- Use subtle orange accents, visible hover/focus feedback, and consistent card
  heights.
- Display three columns on desktop, two on tablet, and one on mobile.
- Leave all surrounding sections unchanged.
