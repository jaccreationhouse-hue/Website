# What You Can Expect Heading Consistency Design

## Goal

Bring the What You Can Expect marquee heading into the same visual hierarchy as
the Industries, Highlights, and What We Do sections without changing marquee
behavior.

## Approved Design

- Add chip: `What We Promise`
- Keep title: `What You Can Expect`
- Add concise supporting copy
- Reuse the shared `home-section-heading` pattern
- Preserve marquee content, direction, pause behavior, and reduced-motion mode
